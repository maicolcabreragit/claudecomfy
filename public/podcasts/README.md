# Podcast Audio Files

This directory stores generated podcast audio files.

## File Naming Convention

- `ep{number}-{title-slug}.mp3` - Full episodes
- `preview-{hash}.mp3` - Temporary preview files

## Specifications (Spotify-compatible)

- Format: MP3
- Bitrate: 128 kbps
- Sample Rate: 44.1 kHz
- Channels: Stereo

## Cleanup

Preview files older than 24 hours may be automatically deleted.
